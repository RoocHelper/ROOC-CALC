⚔️ ROOC Damage Simulator & Calculator

Kalkulator interaktif berbasis web untuk menghitung estimasi Damage Per Hit dan DPS (Damage Per Second) di game Ragnarok Origin Classic (ROOC). Alat ini dirancang untuk membantu para pemain meracik dan melakukan theorycrafting build karakter terbaiknya tanpa harus membuang zeny atau material di dalam game.

✨ Fitur Utama

Dual Mode (PvE & PvP): Perhitungan terpisah yang menyesuaikan mekanik damage vs Monster atau adu mekanik vs Pemain.

Simulasi Penetrasi DEF: Kalkulasi akurat sisa damage yang berhasil menembus proteksi PDEF/MDEF target.

Kalkulator DPS: Menghitung jeda serangan (total delay) berdasarkan kombinasi Cast Time, Cooldown, dan Global CD.

Range Fluktuasi (±10%): Menampilkan rentang estimasi damage Minimum dan Maksimum sesuai dengan sistem Weapon Variance di Ragnarok.

🚀 Cara Penggunaan

Kalkulator ini berdiri sendiri (standalone) tanpa memerlukan instalasi aplikasi tambahan.

Unduh atau clone repositori ini.

Buka file Kalkulator_ROOC.html menggunakan browser pilihanmu (Chrome, Edge, Safari, dll).

Masukkan angka sesuai dengan status karaktermu di dalam game. Hasil akan langsung terhitung secara otomatis (real-time).
